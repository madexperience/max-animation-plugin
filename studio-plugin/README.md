# Studio Plugin

The Studio plugin should keep the existing Blender Animations Plugin workflow and UI structure, while replacing user-facing Blender labels with Max labels.

Initial rule:

- Keep the localhost protocol compatible.
- Keep endpoint names unchanged until the Max server passes import/export tests.
- Change UI copy only after the transport is verified.

Expected future contents:

- Rojo project
- Studio plugin source
- Build instructions
- Local plugin install instructions
